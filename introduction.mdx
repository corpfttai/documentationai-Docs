## Quick Start

Get up and running with AdaptAI in minutes. Follow these steps to deploy your first AI support agent.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create your account at `https://dashboard.example.com/signup`. Verify your email to access the dashboard.
  </Step>
  <Step title="Configure API Key" icon="key">
    Generate an API key in the dashboard settings. Store it securely as `YOUR_API_KEY`.

    <CodeGroup tabs="JavaScript,Python">
      ```javascript
      const adaptAI = new AdaptAI({ apiKey: 'YOUR_API_KEY' });
      const response = await adaptAI.query('How do I reset my password?');
      console.log(response.answer);
      ```
      ```python
      from adapt_ai import AdaptAI
      client = AdaptAI(api_key='YOUR_API_KEY')
      response = client.query('How do I reset my password?')
      print(response.answer)
      ```
    </CodeGroup>
  </Step>
  <Step title="Integrate with Tools" icon="plug">
    Use webhooks or SDKs to connect AdaptAI to your chat apps. Test queries in the dashboard playground.
  </Step>
  <Step title="Monitor Analytics" icon="bar-chart-3">
    Review initial interactions in the analytics tab. Refine your knowledge base based on real usage.
  </Step>
</Steps>
