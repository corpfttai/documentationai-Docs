## Implementing 24/7 AI-Powered Answers

Set up AI-powered answers to handle employee queries instantly. AdaptAI uses natural language processing to match questions against your knowledge base.

<Steps>
  <Step title="Create Knowledge Base" icon="database">
    Upload your HR policies, FAQs, and guides.

````markdown
```bash
curl -X POST https://api.example.com/v1/knowledge-bases \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"name": "Employee Handbook", "documents": ["policy.pdf"]}'
```
````

  </Step>
  <Step title="Configure AI Agent" icon="bot">
    Define response styles and escalation rules.

    <CodeGroup tabs="JavaScript,Python">
````javascript
const agent = await adaptAI.createAgent({
  name: 'HR Support',
  knowledgeBaseId: 'kb_123',
  maxTokens: 500
});
````
````python
agent = adapt_ai.create_agent(
    name="HR Support",
    knowledge_base_id="kb_123",
    max_tokens=500
)
````
    </CodeGroup>
  </Step>
  <Step title="Embed Chat Widget" icon="code">
    Add the widget to your intranet or Slack.

    ```html
    <script src="https://cdn.example.com/adaptai.js"></script>
    <adaptai-chat agent-id="agent_456"></adaptai-chat>
    ```
  </Step>
</Steps>

<Callout kind="tip">
  Test your AI agent with sample queries before going live to ensure accuracy.
</Callout>

## Managing Secure Data Storage and Compliance

AdaptAI stores all interactions securely with encryption at rest and in transit. You maintain full control over data retention and access logs.

<Expandable title="Advanced Security Configuration" default-open="false">

Enable compliance features like GDPR export and audit trails.

```javascript
const config = await adaptAI.updateSecurity({
  encryption: 'aes-256',
  retentionDays: 365,
  auditLogs: true
});
```

Key parameters:

<ParamField path="encryption" param-type="string" required="true">
  Choose `aes-256` or `aes-512`.
</ParamField>

<ParamField header="X-Compliance-Region" param-type="string">
  Set to `EU` or `US` for regional compliance.
</ParamField>

</Expandable>

<Callout kind="alert">
  Always use HTTPS endpoints and rotate your `API_KEY` every 90 days.
</Callout>
