## Upgrading to the Latest Version

Follow these steps to ensure your AdaptAI instance runs the newest version. Back up your configuration first.

<Steps>
  <Step title="Backup Configuration" icon="database">

    Export your settings from the dashboard.

````bash
curl -X GET "https://api.example.com/v2/config/export" \
  -H "Authorization: Bearer YOUR_TOKEN"
````

  </Step>
  <Step title="Update SDK" icon="download">

    Update your client SDK across platforms.

    <Tabs>
      <Tab title="npm (JavaScript)" icon="package">
````bash
npm update @adapt-ai/sdk@latest
````
      </Tab>
      <Tab title="pip (Python)" icon="python">
````bash
pip install --upgrade adapt-ai-sdk
````
      </Tab>
    </Tabs>

  </Step>
  <Step title="Migrate Auth" icon="shield">
    <CodeGroup tabs="JavaScript,Python">
````javascript
// Old API key auth
const client = new AdaptAIClient({ apiKey: 'YOUR_API_KEY' });

// New JWT auth
const client = new AdaptAIClient({ 
  token: 'YOUR_JWT_TOKEN',
  baseUrl: 'https://api.example.com/v2' 
});
````
````python
# Old
client = AdaptAIClient(api_key="YOUR_API_KEY")

# New
client = AdaptAIClient(
    jwt_token="YOUR_JWT_TOKEN",
    base_url="https://api.example.com/v2"
)
````
    </CodeGroup>
  </Step>
  <Step title="Verify Deployment" icon="check-circle">

    Test a sample query.

````bash
curl -X POST "https://api.example.com/v2/ai-assist" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -d '{"query": "How do I reset my password?"}'
````

  </Step>
</Steps>

<Callout kind="tip">
  After upgrading, check the analytics dashboard for migration insights. Contact support at `support@example.com` for assistance.
</Callout>
